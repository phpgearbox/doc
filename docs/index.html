The Doc Gear

API Documentation Generator using Markdown

Think phpDocumentor but using 100% markdown syntax. The original idea came from Natural Docs - http://www.naturaldocs.org/. So while this package is geared towards PHP, using reflection where appropriate it works with Javascript and even CSS/LESS/SASS stylesheets. Making it the one stop shop for all your projects documentation needs.

How to Install

Installation via composer is easy:

composer require gears/doc:*
